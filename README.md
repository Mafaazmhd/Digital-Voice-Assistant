# Digital-Voice-Assistant
A digital voice assistant that can tell a joke, calculations, weather, open websits, etc.., made as a python project as for assesment 6 for BACSE101 in VIT vellore, done my K Mafaaz Mohammed, Joash Renji Mangalathu, Muhammed Ayaad, Sevak Raj U, Dharvikk Jalakam

make sure to download flutter to run frontend
to run import:
1.flask
2.flask_cors
3.re (Python standard library)
4.datetime (Python standard library)
5.feedparser
6.requests
7.wikipedia
8.concurrent.futures (Python standard library)
9.urllib.parse (part of the urllib standard library)
10.PyDictionary
11.bs4 (commonly associated with the beautifulsoup4 package)
12.webbrowser (Python standard library)
12.pywhatkit
13.flutter/material.dart (The core Flutter UI framework)
14.speech_to_text (For speech recognition functionality)
15.http (For making network requests)
16.dart:convert (A Dart standard library for encoding/decoding JSON, used in conjunction with http)
17.flutter_tts (For Text-to-Speech functionality)

inorder to connect the flask backend to the flutter frontend copy the ipv4 number form running "ipconfig" in your terminal and paste it in line 188 of the main.dart code
